# 🚀 Quick Start Guide - Fluid Mechanics Calculator v1.1

## ⚡ 5-Minute Setup

### Step 1: Install Python
Download Python 3.8+ from [python.org](https://python.org)

### Step 2: Install Dependencies
Open terminal/command prompt in the project folder:
```bash
pip install -r requirements.txt
```

### Step 3: Run the Application
```bash
python Fluid_Mechanics_Calculator_v1.1_Enhanced.py
```

That's it! 🎉

---

## 🎯 First Time Use

1. **Splash Screen** appears → Wait 2-3 seconds
2. **Main Dashboard** opens → Choose a module
3. **Click a card** to open that calculation module
4. **Start calculating!**

---

## 🎨 Interface Basics

### Main Controls
- **🌙/☀️ Button**: Switch between dark/light themes
- **ℹ️ Button**: View app information
- **Module Cards**: Click to open each chapter

### Module Selection
- 💧 **Fluid Properties** → Density, viscosity, surface tension
- ⚖️ **Fluid Statics** → Pressure, buoyancy, forces
- 🌀 **Fluid Dynamics** → Flow, Bernoulli, energy
- 📐 **Dimensional Analysis** → Similitude, scaling

## 🆘 Need Help?

- **Can't install?** → Check Python version: `python --version`
- **Module won't open?** → Ensure all files are in same folder
- **Calculation error?** → Verify units are SI (m, kg, s)
- **More help?** → See USER_GUIDE.md
